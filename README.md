# MAPPO-Based Energy-Efficient Joint Access and Backhaul Resource Allocation Framework for 5G HetNets

## Overview

This project proposes a MAPPO-based framework for energy-efficient resource allocation in 5G heterogeneous networks using ns-3 and 5G-LENA.

## Features

* MAPPO and PPO based optimization
* Dynamic SBS sleep mode
* Energy efficiency optimization
* Backhaul-aware resource allocation
* Throughput, SINR and Delay analysis

## Tools Used

* ns-3.38
* 5G-LENA
* Python
* PyTorch
